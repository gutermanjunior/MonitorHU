# Monitor HU USP

Programa em Python para monitorar vagas de especialidades no Hospital Universitário.

## Instalação

1. Instale as dependências:
   pip install -r requirements.txt

2. Configure o ambiente:
   - Renomeie o arquivo `.env.example` para `.env`
   - Preencha com suas senhas e tokens reais dentro do `.env`

3. Rodar:
   python MonitorHU.py
